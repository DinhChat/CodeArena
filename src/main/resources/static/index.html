CodeArena API Platform

This server provides the backend API for CodeArena, an academic-oriented online programming judge designed for automated code evaluation, submission processing, and secure execution of source code.

  • RESTful APIs for problem management, test cases, and submissions
  • JWT-based authentication and role-based authorization
  • Isolated code execution with memory and time constraints
  • Designed for scalability, security, and academic use

To access the CodeArena web interface, please proceed to the official frontend application.
